# Day 16 - Node.js Basics

## Setup
npm install

## Scripts
- `node hello-node.js` — prints Node version, file, dir, and repeating messages (stops after 10s).
- `npm start` — runs banner.js (prints figlet + chalk banner).
- `node greet.js <Name>` — greets the user with current date/time.

Optional:
- `node greet-moment.js <Name>` — same as greet but uses moment for formatting.
